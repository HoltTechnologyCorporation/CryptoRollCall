CryptoRollCall Public APIs (Alpha)
 
List of available APIs
  • GET - /ticker/list
    Retrieves the entire list of crypto currency tickers
  • GET - /ticker/(id)
    Retrieves the crypto currency ticker for the specified id.
  • GET - /currency/(currency)
    List of exchange rates based on the base currency.
  • GET - /charts/histo/(period)/(id)
    Retrieves the chart data for the top performing coins in the previous hour, previous day and previous week.
  • GET - /coindetail/(symbol)
    Retrieves all the details for the specified coin, including social links, source code repositories and available exchanges
  • GET - /charts/home
    Retrieves the chart data for the top performing coins in the previous hour, previous day and previous week.
  • GET - /coindetail/(symbol)
    Retrieves all the details for the specified coin, including social links, source code repositories and available exchanges

For more detailed information, please visit https://app.swaggerhub.com/apis/cryptorollcall/CryptoRollCall/1.0.0
