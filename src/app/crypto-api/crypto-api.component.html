CryptoRollCall Public APIs (Alpha)

For detailed information, sample requests and responses, field descriptions and to try it out for free, please visit https://app.swaggerhub.com/apis/cryptorollcall/CryptoRollCall/1.0.0

List of available APIs
  • GET - /ticker/list
    Retrieves the entire list of crypto currency tickers
    Example GET url : https://www.cryptorollcall.com/crcserver/ticker/list

  • GET - /charts/histo/(period)/(id)
    Retrieves the chart data for the top performing coins in the previous hour, previous day and previous week.
    Example GET url : https://www.cryptorollcall.com/crcserver/charts/histo/DAY/BTC
    Example Response :

                        }
                        | json
                 }}
